to use the program:
Run these classes by this order:
 1-SpiderTest
 2-Ranker_test
 3-Indexer_test
Now, you have your database ready, and you can go to interface to search.
And for recrawling & reindexing, you can run MAIN_SEARCH_ENG and search for query from the console.
